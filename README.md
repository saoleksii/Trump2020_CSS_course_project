# Info about project
The aim of the project is to analyse US elections 2020 and capytol attack using dataset from reddit.
Folder scripts contains scripts used for this project.
select_data.py is used to select and preprocess dataset, merge all csv files into one.
combine_files.py is used to merge 2 preprocessed csv files.
Main part of the preprocessing is in Trump2020_CSS_course_project.ipynb file:
